BUILD INSTRUCTIONS:
1.Make sure Miniconda is installed
2.run OpenMiniConda batch file to open miniconda command prompt
3.run SetupCondaEnv batch file inside the command prompt to create the project's anaconda environment

USING THE PROJECT:
1.run OpenMiniConda batch file to open miniconda command prompt
2.run ActivateCondaEnv batch file inside the command prompt to activate the project's anaconda environment
3.run launcher.py with python inside the anaconda environment to run the console
